# はじめに

**不思議の国のアリス**（原題：*Alice's Adventures in Wonderland*）は、イギリスの作家ルイス・キャロル（チャールズ・ドジソンの筆名）によって1865年に書かれた小説です。うさぎの穴を通って不思議な世界に落ちた、アリスという名前の少女の物語です。

この物語は論理を使って遊んでおり、子供だけでなく大人にも愛され続けています。ナンセンス文学の最高傑作の一つとされています。

## この版について

この版には、元の作品から選ばれた章が含まれています：

- **第1部：うさぎの穴を下って** - アリスの不思議の国への降下と最初の出会い
- **第2部：不思議の国の冒険** - 不思議な住人たちとのアリスの冒険

原作は1865年11月26日に出版され、現在はパブリックドメインとなっています。
