# 第2章：涙の池

「なんて奇妙なことでしょう！」とアリスは叫びました（彼女はとても驚いていたので、その瞬間、正しい英語を話すことを完全に忘れていました）。「今度は望遠鏡のように伸びているわ！さようなら、私の足！」（彼女が足を見下ろすと、足は遠くに行ってしまったように見えて、ほとんど見えなくなっていました）

## サイズの変化

アリスは自分がどんどん大きくなっていくのを感じました。数分で彼女はとても背が高くなり、床に横にならなければなりませんでした。さらに1分が経過し、もはや部屋に収まらなくなりました。

「これ以上大きくならないように願います」とアリスは思いました。「もうドアを通り抜けることができません。こんなに大きくならなければよかったのに！」

しかし、彼女は大きくなり続けました。最後の手段として、アリスは片腕を窓から出し、片足を煙突に入れました。「これ以上は何もできない」と彼女は思いました。「どうなってしまうのかしら？」

## 小さくなる

幸いなことに、小さな魔法の瓶の効果が止まり、アリスはこれ以上大きくなりませんでした。しかし、それは非常に不快でしたし、この部屋から出る望みがないように思われたので、アリスはとても不幸でした。

「家にいた方がずっとよかったわ」とかわいそうなアリスは思いました。「絶え間なく大きくなったり小さくなったりせず、ネズミやウサギに命令されることもなかったのに」
